# Contributing to Social Starter Pack

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Clone locally
3. Run `make install` to set up all tools
4. Run `make check` to verify your setup

## Development

- Each tool lives in `packages/<tool-name>/`
- Run tests: `make test`
- Each package has its own CI workflow

## Pull Requests

- One feature/fix per PR
- Include tests for new features
- Update relevant documentation

## Ideas for Contributions

- New platform integrations (Bluesky, Mastodon, etc.)
- Additional output formats
- Documentation improvements
- Bug fixes
